# Approval System

一套以企業內部簽核流程為核心的全端系統。

---

# DEMO 網址

http://approvalsystem.sytes.net/

---

# 專案功能

- 權限區分
- 申請單 CRUD 管理
- Email寄送 (Hangfire 背景任務)
- 即時 Log 監控（SignalR）
- CI/CD 自動部署 (Github Action)
